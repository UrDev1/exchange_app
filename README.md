Простой конвертер валют - веб-приложение, позволяющее пользователям 
быстро пересчитывать суммы между различными валютами по актуальному курсу. 
Интерфейс интуитивно понятный, с минимальным количеством элементов управления.

Как запустить 

 pip install -r requirements.txt /n
 cd app/ /n
 ./manage.py migrate /n
 ./manage.py runserver
